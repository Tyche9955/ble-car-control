# BLE 小车控制 APP

基于 Flutter 开发的蓝牙低功耗（BLE）小车无线遥控应用，支持 Android 手机通过 BLE 模块控制小车运动、灯光和蜂鸣器。

---

## 功能特性

- 🔍 **BLE 扫描** — 自动扫描周边蓝牙设备，显示信号强度（RSSI）
- 🔗 **一键连接** — 点击设备即可连接，自动发现 UART 特征值
- 🕹️ **方向控制** — 前进 / 后退 / 左转 / 右转 / 停止
- ⚡ **速度调节** — 滑块实时调速（0–100%）
- 💡 **灯光控制** — 开灯 / 关灯
- 🔔 **蜂鸣器** — 一键鸣笛
- 🔧 **HEX 自定义指令** — 手动输入十六进制指令发送
- 📋 **通信日志** — 实时显示收发数据，方便调试

---

## 支持的 BLE 模块

| 模块类型 | TX 特征 UUID | RX 特征 UUID |
|---------|-------------|-------------|
| Nordic UART (nRF) | `6E400002-...` | `6E400003-...` |
| HM-10 / HC-08 | `FFE1` | `FFE2` |

---

## 指令协议

所有指令格式为 **2 字节**：`[命令码, 速度]`

| 功能 | 命令码 | 示例（速度50%） |
|------|--------|----------------|
| 停止 | `0x00` | `00 32` |
| 前进 | `0x01` | `01 32` |
| 后退 | `0x02` | `02 32` |
| 左转 | `0x03` | `03 32` |
| 右转 | `0x04` | `04 32` |
| 开灯 | `0x10` | `10` |
| 关灯 | `0x11` | `11` |
| 蜂鸣 | `0x20` | `20` |

## 项目结构

```
lib/
├── main.dart           # 应用入口，主题配置
├── car_manager.dart    # 核心逻辑：BLE 连接、指令发送、状态管理
├── scan_screen.dart    # 扫描页面：设备列表
└── control_screen.dart # 控制页面：方向盘、速度、功能按钮
```

## 环境要求

- Flutter 3.22.3+
- Android 5.0（API 21）及以上
- 手机需开启蓝牙和位置权限

---

## 本地构建

```bash
# 安装依赖
flutter pub get

# 调试构建
flutter build apk --debug

# 正式构建
flutter build apk --release
```
